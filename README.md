## :rocket: Rodando o projeto

As questões podem ser testadas das seguintes formas:

- Com uma IDE na máquina local para abrir os arquivos.
  - [IntelliJ](https://www.jetbrains.com/help/idea/creating-and-running-your-first-java-application.html#write-code).
  - [Eclipse](https://www.javatpoint.com/how-to-run-java-program-in-eclipse).
  
- Por um compilador Online, copiando as linhas de código ou colocando os arquivos .java das questões.
  - [Online Java Compiler - JDOODLE.](https://www.jdoodle.com/online-java-compiler/)
  - [Online Java Compiler - OnlineGDB.](https://www.onlinegdb.com/online_java_compiler)
  
- Utilizando a linha de comando:
  - Necessário git instalado.
  - Necessário o JDK LTS instalado.
  - Exemplo feito no Microsoft PowerShell para abrir a questão 01.
  
```bash

git clone https://github.com/bruscapin/desafio-2-bruna-scapin.git

cd .\desafio-2-bruna-scapin\src\

javac Questao01.java

java Questao01

```
